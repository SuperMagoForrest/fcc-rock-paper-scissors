# Rock Paper Scissors (freeCodeCamp Project)

This project is part of the freeCodeCamp Machine Learning curriculum.

- `RPS_game.py` → game engine (do not modify).
- `main.py` → to run local matches.
- `test_module.py` → unit tests.
